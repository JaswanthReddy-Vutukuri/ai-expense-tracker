# Frontend Application (Angular)

Purpose:
- User interface for Expense Tracker
- Expense CRUD UI
- Dashboard UI
- AI Chat UI

Rules:
- Use Angular best practices
- No backend or AI logic
- Communicate via HTTP only
- Backend API base URL is configurable
- AI Orchestrator is accessed via /ai/chat

Do not:
- Call LLMs directly
- Contain API secrets
- Implement business logic
